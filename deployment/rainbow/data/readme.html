Windows Platform

Requirements

In order to run Rainbow on Windows you must have a running installation of Java 1.5 or later.

Installation

To install Rainbow on Windows:

  1. Download the ZIP file of the Windows distribution.
  2. Unzip the file on any location, we recommend C:\Program Files\Rainbow6.

To run Rainbow: simply double-click on the rainbow.exe file in the main folder.

You can also run the application with the following command line (assuming it has been unzipped in the recommended folder):

C:\>javaw -jar C:\Program Files\Rainbow6\lib\rainbow.jar

Uninstallation

To uninstall Rainbow

  1. Rainbow runs independently of Windows, so to uninstall it you can simply delete the folder where you have installed it. There is no registry dependencies to remove.
  2. If you want you can also remove the file where the user preferences are saved. It is located in the Home folder of the user and called .Rainbow.

Linux Platforms

Requirements

In order to run Rainbow on Linux you must have a running installation of Java 1.5 or later.

Installation

To install Rainbow on Linux (32- and 64-bit):

  1. Download the ZIP file of the Linux distribution you need (there is one for 32-bit and one for 64-bit).
  2. Extract the GunZip file on any location.
  3. Set the execution rights for the file called rainbow.sh on the main directory.

To run Rainbow: simply double-click on the rainbow.sh file in the main directory.

You can also run the application from the command line (assuming Rainbow is installed in \home\archie\Rainbow6):

archie@archie:~>java -jar Rainbow6/lib/rainbow.jar

Uninstallation

To uninstall Rainbow:

  1. Delete the directory where Rainbow is installed.
  2. If you want, you can also remove the file where the user preferences are saved. It is located in the Home folder of the user and called .Rainbow.

Macintosh Platform

Requirements

On both Mac OS X 10.5 (Leopard) or higher, and Mac OS X 10.4 (Tiger), Rainbow requires Java 1.4.2 or Java 1.5.x. Note that Java 1.6 (which is 64-bit) is not supported yet. Currently Rainbow uses Carbon, not Cocoa. Support for Cocoa will come when the Cocoa implementation of SWT is stable.

Installation

To install Rainbow on Macintosh:

  1. Download the ZIP file of the Macintosh distribution.
  2. Extract the file on any location. For example /Users/admin/Desktop/rainbow.sh

To run Rainbow:

  1. Try double-clicking on the rainbow.sh shell script that is in the main directory of the distribution.
  2. If this does not work:
    1. Open a terminal window.
    2. Go to the main directory of the distribution, for example /Users/admin/Desktop/rainbow
    3. Execute the following command:
    java -XstartOnFirstThread -jar lib/rainbow.jar

You MUST specify the -XstartOnFirstThread option or Rainbow may not work properly.

Uninstallation

To uninstall Rainbow:

  1. Delete the directory where Rainbow is installed.
  2. If you want, you can also remove the file where the user preferences are saved. It is located in the Home folder of the user and called .Rainbow.
