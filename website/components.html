Filters

Some of the most important type of component in the framework are the filters. They allow you to read a given document and get access to its content using the same API regardless of the file format.

Currently the following filters are available:

  • HTML Filter - A filter for HTML and XHTML files.
  • XML Filter - A filter for XML documents.
  • Properties Filter - A filter for properties files.
  • Regex Filter - A filter for different type of text-based files. This filter uses customizable regular-expression rules to define the parts of the document that need to be treated as translatable text. It comes with pre-defined parameters for the following formats.
  • XLIFF Filter - A filter for XLIFF documents.

Other filters are being developed.

  • OpenXML Filter - A filter for Microsoft Office files such as DOCX, XLSX and PPTX.
  • OpenOffice.org Filter - A filter for OpenOffice.org files such as ODT, ODS, ODG and ODP.
  • TMX Filter - A filter for the Translation memory eXchange (TMX) file format.
  • IDML Filter - A filter for Adobe In-Design XML documents.
