Support

If you were not able to find an answer to your question in the on-line help:

  • Try the Okapi Tools users group mailing list:
    • See the mailing list archives (public access)
    • Subscribe to the user group
    • Post a message (you need to member of the list)
  • Try the Google Code project site
    • See the list of reported issues (public access)
    • Post a bug reports or a request for feature (you need to have a Google account)
  • Or send a direct email by selecting the Send Feedback command in the Help menu of most Okapi applications.
