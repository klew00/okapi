Okapi Framework

Downloads

Latest Releases

All the packages, current and passed, are accessible from the Download area of the Google Code project site
The latest releases are:

>>> Okapi Applications (okapi-apps) - Version @okapiappsVersion@

This distribution is for users and developers. It includes all the files of the okapi-lib distribution (except the developers documentation), as well as the UI components of the framework, and several applications: Rainbow (version @rainbowVersion@), Ratel (version @ratelVersion@) and Tikal (version @tikalVersion@).

Downloads: Windows | Linux | Linux 64-bit | Macintosh Cocoa | Macintosh Cocoa 64-bit
 

>>> Okapi Library (okapi-lib) - Version @okapilibVersion@

This distribution is for developers. It includes all the core libraries, filters, and other non-UI components of the Okapi Framework. It also includes Tikal (version @tikalVersion@). You want this distribution if you need only the non-UI parts of the framework.

Downloads:  All Platforms (ZIP) | All Platforms (GZIP)
 

Latest Snapshots

The latest snapshot builds are the latest successful builds created from the source code being currently developed. They have the most up-to-date features, but they may be untested and some parts maybe unstable. They are the versions used by the developers. Use them at your own risk.

The latest snapshots are located here: http://okapi.opentag.com/snapshots The dates and times of the files are in US Pacific Time.

Older .NET Releases

Older releases of the .NET implementations of the Okapi Framework , such as Rainbow v5 and Olifant are still available, but are no longer developed.

See this site: http://okapi.sourceforge.net/downloads.html for the downloads of the .NET applications.
