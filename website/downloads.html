Latest Releases

All the packages, current and passed, are accessible from the Download area of the Google Code project site
The latest releases are:

===> Rainbow - Version @rainbowVersion@
Rainbow is a toolbox that provides various utilities to help you in localization tasks. This package includes both the classes that are in the Okapi Library as well as in Ratel.

  • Downloads: Windows | Linux | Linux 64-bit | Macintosh

  • Online Help | Screenshots

===> Okapi Library - Version @okapilibVersion@
The Okapi Library is a Java library with all the classes needed to use the Okapi filters components with your own program and scripts.

  • Download:  Windows | Linux | Linux 64-bit | Macintosh

  • Developer's Guide: Getting Started | Table of Contents

===> Ratel - Version @ratelVersion@
Ratel is a program to create and maintain segmentation rules in the SRX (Segmentation Rule eXchange) format. (Note: the functionality of Ratel is included in the Rainbow distribution)

  • Downloads: Windows | Linux | Linux 64-bit | Macintosh
  • Online Help | Screenshots

Older .NET Releases

Older releases of the .NET implementations of the Okapi Framework , such as Rainbow v5 and Olifant are still available, but are no longer developed.

See this site: http://okapi.sourceforge.net/downloads.html for the downloads of the .NET applications.
