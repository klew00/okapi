Requirements

In order to run Rainbow you must have a running installation of Java 1.5 or later.

You can get Java for free from different places:

  • Sun's Java: http://www.java.com/en/download/
  • OkpenJDK: http://openjdk.java.net/
  • more Java VMs are listed here: http://en.wikipedia.org/wiki/List_of_Java_virtual_machines

Installation and Uninstallation

Windows

To install Rainbow on Windows (32-bit):

  1. Download the ZIP file of the Windows ditribution: LocationTODO
  2. Unzip the file on any location, we recommend C:\Program Files\Rainbow6.

To run Rainbow: simply double-click on the rainbow.exe file in the main folder.

You can also run the application with the following command line (assuming it has been unzipped in the recommended folder):

C:\>javaw -jar C:\Program Files\Rainbow6\lib\rainbow.jar

To uninstall Rainbow

  1. Rainbow runs independantly of Windows, so to uninstall it you can simply delete the folder where you have installed it. There is no registry dependencies to remove.
  2. If you want you can also remove the file where the user preferences are saved. It is located in the Home folder of the user and called .Rainbow.

Linux

To install Rainbow on Linux (32-bit):

  1. Download the ZIP file of the Linux ditribution.
  2. Extract the GunZip file on any location.
  3. Set the execution rights for the file called rainbow on the main directory.

To run Rainbow: simply double-click on the rainbow file in the main directory.

You can also run the application from the command line (assuming Rainbow is installed in \home\archie\Rainbow6):

archie@archie:~>java -jar Rainbow6\lib\rainbow.jar

To uninstall Rainbow:

  1. Delete the directory where you had installed Rainbow.
  2. If you want you can also remove the file where the user preferences are saved. It is located in the Home folder of the user and called .Rainbow.
