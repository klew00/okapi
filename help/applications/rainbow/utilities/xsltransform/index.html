Okapi Framework - Utilities

XSL Transformation
Utilities Okapi Framework

This utility allows you to apply a given XSL Transform template to an input document and generate the resulting output.

This utility does not use filters.

Input Parameters

  • The list of the XML documents to transform (Input list 1).
  • The names and locations of the output documents.

Configuration

Options Tab

Path of the XSLT template to apply -- Enter the full path of the XSLT template to use to generate the output. You can use the variable ${ProjDir} to specify the directory of the project.

Parameters -- Enter the list of any required parameters for this template. the format must be name=value with one parameter per line. For example:

Lang=en
tmFile=myTM.tmx

Default Parameters -- Click this button to copy into the parameters list all the default parameters declared in the template.

Open Template -- Click this button to open the template file.
