Utility – Used Characters Listing

Overview

This utility allows you to get a list of all characters used in the translatable text of a give set of files.

This is useful for example when you need to create a font for bitmap Japanese characters for a game interface. You need to include in the font only the characters actually used in the text, so you can reduce the size of the bitmap font.

The result file is a tab-delimited text file with three columns.

  1. The Unicode code point
  2. The actual character
  3. The number of occurences found.

For example:

U+2192 '→' 50
U+00AE '®' 1
U+00A0 ' ' 3
U+007A 'z' 10
U+0079 'y' 320
...

The output list is not sorted.

Input Parameters

  • The list of the documents to process (Input list 1).
  • The default input encoding.

Configuration

Options Tab

Path of the result file -- Enter the full path of the result file. (Configuration key = outputPath)

Open the result file after completion -- Set this option to automatically open the result file after the process is complete. (Configuration key = autoOpen).
